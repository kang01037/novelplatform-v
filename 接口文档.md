# 小说平台接口文档

## 1. 接口概述

本文档描述了小说平台的所有API接口，包括用户管理、小说管理、章节管理、书架管理和评论管理等功能模块。

### 1.1 基础信息

- 接口基础路径：`/api`
- 响应格式：JSON
- 跨域支持：`@CrossOrigin(origins = "*")`

### 1.2 响应结构

所有接口返回统一的响应结构：

```json
{
  "code": "响应码",
  "message": "响应消息",
  "data": "响应数据"
}
```
