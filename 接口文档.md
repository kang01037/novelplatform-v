# 小说平台接口文档

## 1. 文档说明

本文档详细描述了小说平台的所有后端接口，包括用户、小说、章节、书架、评论和微信登录模块。所有接口均基于RESTful风格设计，使用JSON格式进行数据交互。

## 2. 响应格式

所有接口的响应格式统一为：

```json
{
  "code": "200",
  "message": "操作成功",
  "data": "返回数据"
}
```

- `code`: 响应状态码，200表示成功，其他表示失败
- `message`: 响应消息，描述操作结果
- `data`: 响应数据，根据接口不同返回不同格式的数据
